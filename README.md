# ProyectoAngular

Fullstack application built with Angular 21 + NestJS + PostgreSQL (Supabase).

## Stack

- **Frontend**: Angular 21, Tailwind CSS, TypeScript
- **Backend**: NestJS, TypeORM, JWT Auth
- **Database**: PostgreSQL (Supabase)

## Setup

### Backend

```bash
cd backend
npm install
# Configure .env with your Supabase credentials
npm run start:dev
```

### Frontend

```bash
cd frontend
npm install
ng serve
```
